# Упрощенная версия потокового редактора sed
Программа умеет выполнять базовые операции редактирования текста на основе регулярных выражений

## Описание программы
Программа выполняет следующие операции:
1. **Замена всех вхождений указанного текста на другой текст.**
2. **Удаление всех строк, соответствующих заданному регулярному выражению.**
3. **Добавление указанного текста в начало или конец каждой строки.**

### Примеры запуска программы:
```bash
./sed_simplified input.txt 's/old_text_regex/new_text/'
./sed_simplified input.txt '/regex/d'
./sed_simplified input.txt 's/^/prefix/'
./sed_simplified input.txt 's/$/suffix/'
```
